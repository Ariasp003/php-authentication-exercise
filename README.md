# php-authentication-exercise
PHP authentication system (Lab exercise)

This project demonstrates a basic PHP authentication system using MySQLi.

## Features
- User login authentication
- Session management
- Protected pages
- Add user record
- View registered users
- Error message on invalid credentials

## Technologies
- PHP
- MySQL (MySQLi)
- XAMPP
- HTML & CSS

## How to Run
1. Import the database using the SQL script.
2. Start Apache and MySQL in XAMPP.
3. Open `login.php` in your browser.
